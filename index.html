🚀 Panduan Setup Langkah demi Langkah

1
Buka Google Sheets

Buat Google Sheets baru. Beri nama misalnya "Data Laporan Privat". Catat URL-nya.

2
Buka Apps Script

Di Google Sheets, klik menu Ekstensi → Apps Script. Hapus kode yang ada, lalu paste seluruh kode dari file backend.gs yang sudah disertakan.

3
Ganti ID Sheet & Folder Drive

Di baris paling atas kode Apps Script, isi:

const SHEET_ID = "ID_GOOGLE_SHEET_ANDA";
const FOLDER_ID = "ID_FOLDER_DRIVE_ANDA";

ID Sheet ada di URL Sheets: docs.google.com/spreadsheets/d/[INI ID-NYA]/edit

ID Folder Drive: buka folder Drive → lihat URL: drive.google.com/drive/folders/[INI ID-NYA]

4
Deploy sebagai Web App

Klik Deploy → New deployment. Pilih tipe: Web app. Atur:

Execute as: Me
Who has access: Anyone

Klik Deploy → izinkan akses. Salin Web App URL yang muncul.

5
Paste URL ke Tab Setup

Kembali ke aplikasi ini, buka tab ⚙️ Setup, paste URL tadi, klik Simpan URL. Lalu klik Test Koneksi untuk memastikan berhasil.

6
Hosting File HTML (Opsional tapi Direkomendasikan)

Upload file app.html ini ke:

• GitHub Pages (gratis)
• Netlify Drop (gratis)
• Google Drive → "Bagikan" → salin link

Setelah punya URL permanen, link orang tua bisa langsung dibagikan.

7
Tambah Siswa & Mulai Input!

Buka tab 👦 Data Siswa → tambah nama siswa. Kemudian buka tab ✏️ Tentor → isi laporan setiap pertemuan. Link orang tua tersedia di tab 👨‍👩‍👧 Orang Tua.
